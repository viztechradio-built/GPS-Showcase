## üéØ Overview

GPS-Showcase is a layer solution software that lives on top of Google Maps Street View, providing users with a more personalized traveling experience. Users can browse restaurants, bars, and other establishments' ambiances in their area before selecting a route, making informed decisions about their dining and entertainment choices.

**Directed by James Gordon Watkins**

## üöÄ What Problem Does GPS-Showcase Solve?

GPS-Showcase bridges the gap between tourists and their experiences when visiting new environments. It addresses:

- **Discovery Challenge**: Finding restaurants that match your taste and style
- **Ambiance Uncertainty**: Seeing the atmosphere before visiting
- **Decision Fatigue**: Too many options without context
- **Local Experience**: Connecting travelers with authentic local establishments

## üì¶ Installation

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Web server (for local development)
- Google Maps API Key (for full functionality)

### Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/viztechradio-built/GPS-Showcase.git
   cd GPS-Showcase
   ```

2. **Open with a local server**
   
   Using Python:
   ```bash
   python -m http.server 8000
   ```
   
   Using Node.js (http-server):
   ```bash
   npx http-server -p 8000
   ```
   
   Using PHP:
   ```bash
   php -S localhost:8000
   ```

3. **Access the application**
   ```
   Open your browser and navigate to: http://localhost:8000
   ```
